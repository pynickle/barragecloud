# barragecloud

## 调试

```bash
python app.py
```

## 目录
```directory
- app.py: 主文件
- instance: 实例文件夹
  - config.py: 配置文件
- templates
  - user: 用户信息
  - video: 视频信息
  - index.html: 主页面
  - help.html: 帮助文件
- static: 静态文件
  - user
  - video
  - help
- src: python文件
  - b_*: 蓝图文件
  - api: 获取信息
```
